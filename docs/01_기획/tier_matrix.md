# Odab-Note 서비스 티어 매트릭스 (Tier Matrix)

로컬 설치형 오픈소스 기반의 핵심 CLI 기능은 완전히 무료로 개방하며, 수많은 AI 에이전트 개발 과정에서 누적 정제된 **"비공개 고밀도 오답노트 백신 프리셋"**과 **자동화 마이닝 시뮬레이터**는 유료 구독(SaaS) 및 기업용 엔터프라이즈 플랜으로 판매합니다.

## 2. 티어별 세부 가치 제안 (Value Proposition)

### 1) Free Tier (개인용 로컬 에디션)
* **목적:** 진입 장벽을 완전히 제거하고 수만 명의 개발자가 매일 쓰는 개발 표준으로 자리 잡기 위함.
* **사용 방식 및 데이터 헌납:** 로컬 PC에서 독자적으로 `odab-note` 데이터베이스를 무상으로 운용하는 대신, 개발 과정에서 수집되고 매칭 성공한 오답노트 로그(에러 패턴 및 매칭된 해결 코드)를 당사의 분석 플랫폼으로 **자동 익명 전송(Telemetry)**하여 백신 프리셋 품질 향상에 기여해야 함.

### 2) Pro Tier (상용 AI 에이전트 빌더용)
* **목적:** AI 에이전트 구동률의 획기적인 상승과 시간 절약.
* **제공 혜택 및 보안:** 
  * 매주 크롤링 및 실험을 통해 갱신되는 **Claude 3.5 Sonnet / GPT-4o / Gemini 1.5 Pro 전용 코딩 실수 템플릿(Preset) 공급**.
  * 개인 개발 및 상용 소스 보안을 보장하기 위해, **익명 오답 텔레메트리 전송을 완전히 해제(Opt-out)**할 수 있는 기능 잠금 해제.
  * 여러 장치에서 오답노트를 통합 관리할 수 있는 클라우드 Sync 제공.

### 3) Enterprise Tier (사내 설치형 AI 에이전트 가동 기업)
* **목적:** 사내 보안 환경 하에 구동되는 소형 온디바이스 AI 에이전트의 완성도 극대화 및 지적 자산 완전 은닉.
* **제공 혜택 및 보안:**
  * Llama-3-8B-Instruct, Gemma-2-9B 등 **오픈소스 소형 모델이 코딩 시 자꾸만 문법을 파괴하거나 잘못된 API 호출을 반복하는 버그에 특화된 독점 프리셋 데이터셋** 전면 공급.
  * 기업이 다루는 특정 프레임워크나 내부 API에 맞춘 **오답 채굴 시뮬레이터(Simulator Test-bed) 커스텀 개발 및 자동 데이터 주입 파이프라인** 셋업.
  * **철저한 데이터 은닉:** 어떠한 텔레메트리 통신도 발생하지 않는 완전 폐쇄망 동작 모드 지원.
  * **SQLCipher 암호화:** 로컬 SQLite 파일 자체를 통째로 기기 고유 식별 해시 기반으로 AES-256 암호화하여 로컬 디스크 상의 데이터 유출을 원천 방어.
